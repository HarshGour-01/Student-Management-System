# Student Management System

A console-based Student Management System developed in C using Structures, Functions, and Binary File Handling.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

## Technologies Used

- C Language
- File Handling
- Structures
- Functions

## How to Run

Compile:

```bash
gcc main.c -o student
```

Run:

```bash
.\student
```
